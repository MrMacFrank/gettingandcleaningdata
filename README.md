gettingandcleaningdata
======================

getting and cleaning data - Coursera course


how to get started
------------------
1. mount your workspace by using setwd()
2. download 'run_analysis.R'
3. download raw data stored in .zip-file (use download.file() in 'run_analysis.R')
4. unzip the downloaded .zip-file manually into your workspace
5. run the script 'run_analysis.R'
6. look into your set workspace directory to find "data.final.RData" as the cleaned dataset!
